# Contributing

Thank you for your interest in contributing to django-pgschemas!

## Getting Started

- Join the discussion at https://github.com/lorinkoz/django-pgschemas/discussions.
- PRs are welcome! If you have questions or comments, please use the discussions link above.

## Running Tests

To run the test suite run `mise run test` or `mise run coverage`. The tests for this project live inside a small django project called `sandbox`.

## Development Setup

1. Install dependencies: `mise run install`
2. Start the database: `mise run up`
3. Run tests: `mise run test`
4. Run type checking: `mise run types`
5. View available tasks: `mise tasks`
